# Code of Conduct

This project is intended to be a respectful and constructive space for everyone.

Harassment, discrimination, or abusive behavior are not tolerated. This includes insults, derogatory comments, and personal attacks. This applies to all interactions: issues, pull requests, discussions, and any other communication.

If you encounter a violation of this code or have concerns about the behavior of others, please [report it to the project maintainer](mailto:drachenpapa.dev@gmail.com).
